MATERIAL REQUIRED:-
1. ARDUINO UNO
2. PROGRAMMING CABLE
3. LAPTOP
4. BUZZER
5. INFRA-RED SENSOR
6. JUMPER WIRE

SOFTWARE: - ARDUINO IDE 1.8.19

PROGRAMMING LANGUAGE: - C++

                               BUZZER (-) TO ARDUINO GND<br>

  IR GND TO ARDUINO GND<br>
  IR OUT TO ARDUINO PIN 2<br>
